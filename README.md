# 재활 솔루션 RehabSolution 프로그램
1. 프로젝트 주제 및 목적
  - 본 프로젝트는 누구나 쉽고 간편하게 신체 재활을 돕는 프로그램을 개발

  - Java Swing 기반 어디에서나 간단하게 재활 운동을 하고, 상태를 기록하며, 분석하여 개인 맞춤형 재활 운동을 생성할 수 있는 재활케어 GUI 프로그램

  - 사용자 친화적 인터페이스: Java Swing을 활용해, 복잡한 기능을 최대한 덜어내고, 촤대한 기본에 충실하고 직관적인 UI를 제공하여 누구나 쉽게 사용할 수 있도록 설계

  - 운동 가이드 제공: 텍스트 및 이미지 기반으로 기본적인 재활 운동 방법을 안내

  - 진행 상황 시각화: 사용자의 통증 강도와 수행 기록을 그래프를 직관적으로 확인

  - 데이터 관리: 회원가입 및 로그인 기능을 통해 환자별 프로필을 생성하고, 자신만의 고유한 루틴을 만들 수 있도록 설계

    1-1. 기대효과
      - 환자가 자기 주도적으로 재활 과정을 관리할 수 있음
      - 재활 진행 상황을 쉽게 파악 가능
      - 누구나 쉽게 접근 가능

2. 코드 설치 및 실행 방법
  - GitHub의 URL을 통해 프로젝트 페이지로 접속
  - 상단 메뉴의 "Releases" 또는 "Code -> Download ZIP" 버튼을 클릭하여 최신 버전을 다운로드
  - 다운로드한 압축파일을 원하는 파일에 해제

    2-1. 필수 조건
      - JDK 17이상 설치
      - VScode 설치(또는 다른 IDE)(VScode설치 시 Extention for java 확장 설치)
      - Windows환경
      - mysql 서버 설치 및 실행(비밀번호 생성)
    2-2. 실행 방법
        - 압축풀기를 한 후 VScode에서 압축을 푼 파일을 열기
        - scr/DataBase.java 파일로 이동
        - pw안의 값을 드래그 후 우클릭 -> 모든항목변경 -> 생성했던 비밀번호 입력
        - Main 실행
